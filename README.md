# Amazon IoT Greengrass (amazon-iot-greengrass)
AWS IoT Greengrass is an open source edge runtime and cloud service that helps you build, deploy, and manage intelligent device software. It seamlessly extends AWS cloud capabilities to edge devices, enabling local data processing, ML inference, and messaging even with intermittent cloud connectivity.

### AWS IoT Greengrass API
